Projeto Blog

Blog desenvolvido durante o curso no SENAC PI

Este projeto foi criado como parte das atividades práticas em sala de aula, utilizando Python com Django e um template Bootstrap para front-end responsivo. O sistema inclui autenticação de usuários, criação e gerenciamento de posts e comentários. Foi uma oportunidade importante para aplicar os conceitos aprendidos e integrar front-end e back-end em um projeto completo.

Projeto Gestor de Estoque

Aplicação desenvolvida com Django em sala de aula

Este projeto foi desenvolvido durante as aulas,
utilizando Python com o framework Django. O sistema permite o controle de estoque com cadastro de produtos, movimentações de entrada e saída e relatórios.
A interface foi feita com Bootstrap, garantindo um layout responsivo e funcional.
